## Setup

1. Create virtual environment:
   python -m venv venv

2. Activate:
   Windows: venv\Scripts\activate
   Mac/Linux: source venv/bin/activate

3. Install:
   pip install -r requirements.txt

4. Run:
   streamlit run app.py

## Deploy on Streamlit Cloud
1. Upload to GitHub
2. Go to https://streamlit.io/cloud
3. Select app.py
4. Deploy
